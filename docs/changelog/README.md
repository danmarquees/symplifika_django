## 📊 Versioning

### **Formato de Versionamento**
Seguimos [Semantic Versioning](https://semver.org/):
- **MAJOR.MINOR.PATCH** (ex: 2.1.3)
- **MAJOR**: Mudanças incompatíveis
- **MINOR**: Novas funcionalidades compatíveis
- **PATCH**: Correções de bugs
